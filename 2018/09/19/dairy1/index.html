    创业过程

    其次，开业期间，黄侦探也要开始谈谈她为什么要开这间事务所以及事务所的筹备过程。
    ·WHY
    因为她开启了她漫长的学习道路，并且在学习过程中有很多心得，几乎每日都会分享给KEN老板，于是KEN老板建议她不如开个个人网站，每天记录下学习过程。而且，黄侦探本身就有一个博客（网易博客），但是前段时间她收到通知，网易博客要关闭了。于是，个人网站的筹建迫在眉睫。
    ·CHOICE
    KEN老板给黄侦探提供了三个选择：
    1、虚拟机+ wordpress ；
    2、使用云主机， Linux +静态生成器；
    3、Github + HEXO
    基于只有薄弱的 HTML&CSS 基础，第三种选择是最适合她的。
    ·DO
    网上有特别多教材，跟着教材走，不懂问百度。
    首先注册 GitHub，然后建立一个仓库（注意：如果下载了 GitHub 桌面版，在桌面版建立的仓库是本地仓库），仓库格式: yourname.github.io 。
    接着下载 node.js。
    然后是 HEXO 的部分。
    黄侦探跟着这个教程走，遇到了很多问题，
    第一个是下载 HEXO 没有成功，主要是操作命令不对，应该是 npm install -g hexo –cli ，而不是 npm install -g hexo 。
    第二个是在上传到仓库时，没有修改 _config.yml 文件。

    上面这个是通过登陆 Github 来通过下一步 hexo -d 步骤。
    或者

    这是通过 SSH 密钥来对接。
    第三个是 进行 hexo -d 这一步骤以后会有一个选择 (yes/no)，选择 yes 。
    黄侦探花了一天加一个晚上的时候，才筹备好“侦探事务所”，其中 SSH 以及 域名 这两件事还没有做完。不急！来日方长，一口吃不成胖子！

    章 完成时间 总结
    Web语言：开始了解HTML 2018/9/6 简单的HTML
    认识HTML中的“HT”：深入理解超文本 2018/9/6 文件分类的链接、图片
    创建网页：构建模块 2018/9/6 分清内联元素、块元素 void元素
    Web镇之旅：连接起来 2018/9/8 为&lta&gt元素加target属性打开新窗口
    eg: <a target=”_blank” … >，如果两个链接的target=”example”， 打开第一个链接以后再打开第二个，第二个会会覆盖第一个链接。
    认识媒体：给网页添加图像 2018/9/10 无
    严格的HTML：遵循标准，合乎规范 2018/9/11 无
    添加一些样式：开始学习CSS 2018/9/12 无
    扩大你的词汇量：字体和颜色样式 2018/9/13 老版本IE不支持像素大小
    与元素亲密接触：盒模式 2018/9/13 盒模式是由内容区、内边距、边框和外边距组合。所有元素都可以被当作盒子。
    class对多个元素使用样式；
    id对只有一个元素或者页面上只有一个元素使用样式，id属性是用来唯一地命名元素。
    字体名称中包含空格需要加上引号；
    区分 PC 与手机端，主要靠 media 元素，以及它是依赖设备的实际屏幕还是浏览器的大小靠(max-device-width or min-width ) 、 ( max-width or min-width )这两个属性区分。
    高级网站构建：div和span 2018/9/16 逻辑区就是页面上彼此相关的一组元素。
    div为块级内容创建逻辑划分；
    span为内联内容的逻辑分组。
    链接状态：未访问、已访问、悬停、focus、active
    链接规则顺序：link、visited、hover、focus、active
    布置元素：布局与定位 2018/9/18 流：就是浏览器再页面上摆放HTML元素所用的方法。
    注意区分width（内容区的）与border-width（边框的）。
    流和盒模型，并排放置两个内敛元素，外边距不会叠起而是相加其外边距大小；上下放置两个块元素，其外边距是其中一个最大外边距大小的大小。
    冻结流局：当用户调节屏幕大小时，设计能保持原样。
    凝胶：介于冻结和流体之间。（它会锁定页面中内容区的宽度，不过会讲它在浏览器中居中。）
    绝对定位：可以在页面上给精准地定位元素。
    流中的元素会调节它们的内容来适应浮动元素的边界。不过绝对定位元素对其他元素没有任何影响。
    指定位置不仅可以使用像素，还有百分比。
    CSS表格显示允许在一个有行有列的表格中显示快元素。
    CSS表格显示中，每个单元格会包含一个HTML块元素。
    CSS表格显示只是使用一种类似表格的布局来表现这个结构。
    HTML表格是表格数据。
